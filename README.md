# SO3_Lie_Algebra_and_Lie_Group_Functions
This repository contains the main functions for operating within the Lie algebra and Lie group, especially considering the $SO(3)$ group, the Special Orthogonal Group.

The $SO(3)$ group is defined as

$$SO(3) = \{ \mathbf{R} \in \mathbb{R}^{3 \times 3} \;\Big|\; \mathbf{R}^T\mathbf{R} = \mathbf{I}, \, \det(\mathbf{R}) = 1 \}$$
